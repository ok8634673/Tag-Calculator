🌟 功能特性：

🔍 智能标签识别

精准识别：只统计以 # 开头的字符串作为标签，智能过滤普通文本

多格式支持：自动处理换行、空格、逗号分隔的标签格式

实时统计：输入时即时计算并显示标签数量


🎯 自定义限制管理
灵活设置：默认50个标签限制，可自定义调整为1-200之间的任意数值
视觉高亮：超出限制的标签自动以红色高亮显示，带有脉动动画效果
智能提醒：清晰显示超出数量，提供直观的视觉反馈

📋 剪切板集成
自动读取：页面加载时自动尝试读取剪切板内容（需用户授权）
一键导入：点击"读取剪切板"按钮手动获取内容
智能填充：自动识别并填充标签内容，提升工作效率

📊 数据统计面板
实时数据：动态显示总标签数、设置限制和超出数量
分类展示：标签按状态分类显示，正常与超出限制标签一目了然
忽略文本显示：单独展示已过滤的非标签文本，确保透明性

🎨 现代化界面
响应式设计：完美适配桌面、平板和手机屏幕
美观配色：采用柔和的渐变背景和清晰的视觉层次
直观操作：精心设计的按钮布局和交互反馈

🚀 快速开始
在线使用
复制完整HTML代码到文本编辑器
保存为 tag-counter.html
在浏览器中打开该文件即可使用

直接使用
只需打开支持剪切板API的现代浏览器（Chrome 66+、Edge 79+、Firefox 63+），即可享受全部功能。

基本操作流程

详细步骤

内容输入
直接在文本框中输入或粘贴标签内容
点击"读取剪切板"按钮导入已复制的内容
等待页面自动读取剪切板（首次需授权）

参数设置
在"标签限制"框中输入期望的数值（1-200）
系统默认显示为50，可根据需要调整

结果查看
系统自动统计并显示标签总数
超出限制的标签以红色高亮标记
底部显示统计摘要和操作建议

内容管理
使用"清空"按钮重置所有输入和统计
实时编辑内容，统计结果即时更新
忽略的非标签文本在底部区域显示

输入格式示例
text
#原神 #原神空月之歌 #原神版本前瞻
#瑶瑶 #依赖摇
#genshinimpact #原神混剪

普通文本和标题会被自动忽略
只有#开头的字符串会被统计

⚠️ 注意事项
权限要求
剪切板访问：首次使用时需授权剪切板读取权限
安全限制：部分浏览器在非HTTPS环境下可能限制剪切板功能
用户交互：某些浏览器要求用户主动交互后才能访问剪切板

使用建议
标签格式：确保标签以#开头，不含空格
数量控制：社交媒体平台通常有标签数量限制，合理设置
内容清理：定期使用清空功能，避免数据累积
权限管理：如有隐私顾虑，可拒绝剪切板权限，手动输入内容

已知限制
不支持图片或富文本剪切板内容
某些移动端浏览器可能限制自动读取功能
标签匹配严格以#开头，不支持其他符号
